Log4E 2026

Eclipse plugin for automatic logging statement insertion. Supports multiple logging frameworks with configurable templates, profiles, and keyboard shortcuts.

Installation

  1. Open Eclipse and go to Help > Install New Software...
  2. Click Add... to add a new repository
  3. Enter the following URL:
https://cristinel-grigoras.github.io/log4e2026/
  1. Select Log4E 2026 from the list
  2. Click Next and follow the installation wizard
  3. Restart Eclipse when prompted

Supported Frameworks

FrameworkLogger Type
SLF4Jorg.slf4j.Logger
Log4j 2org.apache.logging.log4j.Logger
JDK Logging (JUL)java.util.logging.Logger

Features

  • Declare logger field with one shortcut
  • Insert log statements at method entry, exit, and catch blocks
  • Log individual variables or entire methods/classes
  • Substitute System.out/err with logger calls
  • Exchange logging framework across files
  • Configurable log levels, templates, and profiles
  • Project-level and workspace-level preferences
  • Preview wizard for reviewing changes before applying
